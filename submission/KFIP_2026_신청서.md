## 프로젝트

### ▶ 한 줄 정의 (Elevator pitch)

**XRP·Family는 한국 거주 외국인 유학생·근로자(특히 동남아 출신)를 위해 본국 가족 송금과 한국 내 결제 사이의 며칠 단위 시간 단절을 XRPL Escrow 기반 "가족 사전 승인 한도"로 해결하는 가족 송금·결제 통합 지갑 서비스입니다.**

### ▶ 핵심 가치 제안 / 차별화 포인트

| 측면 | 기존 (SWIFT + 한국 카드 충전) | XRP·Family | 개선 |
|---|---|---|---|
| 송금 도착 시간 | 2~5일 | **3.2초** | **약 100,000배 단축** |
| 송금 수수료 | 5~10% | **0.5% 미만** | **10~20배 절감** |
| 결제 가능 시점 | 송금 도착 + 카드/페이 충전 추가 1~2일 | **즉시** (XRPL Payment 1 단계) | 추가 단계 0 |
| 한국 신용평가 | 카드 발급 필수, 외국인 진입 장벽 | **불필요** (가족 자금이 담보) | 외국인 포용 |
| 가맹점 정산 | 카드 PG 2~3일 + 2~3% | 즉시 + 0.3% 이내 | 가맹점도 이득 |

또한 한국 대부업법·여신전문금융업법 적용 영역 **밖** — "가족 내 사전 약정 자금 이체"로 분류되어 자본금 10억+ 라이선스 요건 회피 (PoC 단계 면제, 정식 사업화 시 별도 라이선스 또는 PG 파트너십 트랙).

### ▶ 주요 기능 (5개)

1. **부모 한도 잠금** — 베트남 측 부모가 자녀 앞으로 월 한도를 XRPL `EscrowCreate` + crypto-condition으로 사전 약정. 미사용 잔액은 `CancelAfter` 시점에 부모 자동 환원.
2. **자녀 즉시 수령** — 자녀가 보유한 preimage로 `EscrowFinish` 트리거, 5초 안에 잔액으로 이전. 신용평가 없음.
3. **즉시 가맹점 결제** — 가맹점 QR 스캔 또는 빠른 결제 버튼 → XRPL `Payment` 트랜잭션 1건으로 결제 완료. tx hash testnet explorer 검증 가능.
4. **실시간 환율 oracle** — CoinGecko 기반 XRP-KRW 라이브 환율 60초 캐시 (`/api/rate`). 향후 XRPL DEX 직결로 마진 인터널라이즈.
5. **다국어 UI** — 한국어 / 베트남어 토글 (자녀-부모 양쪽 자국어 환경).

## 해결하고자 하는 문제

### ▶ 문제 정의 (현황 · Pain Point)

한국 거주 외국인 유학생·근로자 중 동남아·중앙아 출신 약 50만 명은 한국 신용카드 발급이 사실상 불가능합니다. 본국 가족이 송금한 자금은 SWIFT 기준 평균 **2~5일** 소요 + **5~10% 수수료** 후 한국 은행 계좌에 도착하며, 카드/페이 충전까지 **추가 1~2일**이 더 걸립니다.

그 시간차 동안 발생하는 실생활 불편 사례:

- 베트남 유학생 Minh: 부모가 송금 보낸 직후 친구에게 5만원 빌려 교통비 사용 → 송금 도착 후 친구에게 돌려주는 패턴 반복.
- 캄보디아 근로자 가족: 월말 식료품 비용 부족 시 라면·기숙사 무료 급식으로 대체.
- 외국인 유학생 50% 이상이 매월 1~2회 "결제할 카드 잔액 부족" 경험 (자체 인터뷰 가설, 본 프로젝트 인터뷰 N≥5 보강 예정).

추상적 "불편"이 아닌 **명확한 자금 시간 압축 결손**입니다.

### ▶ 문제의 규모 / 심각성

| 데이터 | 값 | 출처 |
|---|---|---|
| 한국 등록 외국인 (2024) | 102만+ | 법무부 출입국·외국인정책본부 |
| 베트남 국적 (2위) | 26만 | 법무부 동 통계 |
| 동남아 합계 | 약 50만 | 법무부 동 통계 |
| 베트남→한국 유학생 연평균 증가율 | 10%+ | 한국교육개발원 |
| WireBarley 누적 거래액 | 1조원+ | 회사 공식 보도자료 |
| Sentbe 누적 송금 건수 | 100만+ | 회사 공식 보도자료 |

송금 시장 규모(WireBarley·Sentbe 누적 거래액)는 **외국인 송금 카테고리 자체의 검증된 수요**를 증명합니다. 다만 그들은 **송금까지만** 처리하고 한국 측 결제 채널은 별도 — 본 프로젝트의 진입 공간.

### ▶ 기존 솔루션의 한계

| 솔루션 | 한계 |
|---|---|
| **WireBarley / Sentbe / Hanpass** | 송금 도착까지만 처리. 한국 측 결제는 카드 충전 별도 단계 필요. 즉시성 결손 그대로. |
| **시중은행 SWIFT** | 2~5일 + 5~10% 수수료. 절대 경쟁 불가. |
| **토스 / 카카오페이 후불결제·BNPL** | 한국 신용평가 통과 필수. 외국인 진입 장벽. |
| **신용카드 (Visa/Master)** | 한국 신용평가 또는 본국 카드 + 환차 + 수수료 누적. |
| **P2P / 사다리 대출** | 한국 대부업법 적용, 외국인 신용 인프라 부재로 실질 불가. 또한 부모 자금이 담보가 아님. |

### ▶ 본 프로젝트의 접근 차별성

**"가족 자금의 시간 압축"** 으로 프레이밍:

1. **남의 돈 대출 아님** — 부모가 미리 약정한 가족 자금. 한국 대부업법 적용 영역 밖.
2. **3단계 XRPL native flow**:
   ```
   EscrowCreate(부모, condition)
       ↓ 부모가 자녀 앞으로 한도 잠금
   EscrowFinish(자녀, preimage)
       ↓ 자녀가 preimage 제출 → 즉시 수령
   Payment(자녀 → 가맹점)
       ↓ 가맹점 즉시 결제
   ```
3. **신용평가 0**, 가족 사전 승인 한도가 담보.
4. **CancelAfter 자동 환원** — 부모 자금 안전 보장.
5. **두 화면 + 라이브 터미널** — 부모(베트남 측) / 자녀(한국 측) UI 각각 자국어. 랜딩의 hero terminal은 실 XRPL Testnet account_tx 폴링으로 실시간 트랜잭션 흐름 노출.

### ▶ XRPL을 선택한 이유

| 사유 | 근거 |
|---|---|
| **1. Native Escrow + crypto-condition** | 별도 스마트컨트랙트·audit 0. 이더리움 동일 시나리오 구현 시 보일러플레이트 50% 절감 + L1 가스비 0.5~5 USD/tx 회피. |
| **2. 마이크로 페이먼트 수수료** | 트랜잭션 수수료 < $0.0001 (10~12 drops/tx). SWIFT 5~10% 대비 10,000배 저렴. 소액 결제(편의점 15,000원) 경제성 확보. |
| **3. 3~5초 컨센서스 종결성** | 실시간 결제 가능. Solana와 동급 속도이나, escrow + crypto-condition + native DEX 셋을 동시에 갖춘 L1은 사실상 XRPL이 유일. |
| **4. Native DEX** | 향후 VND ↔ XRP ↔ KRW 환전 경로 직결 가능. 별도 환전 인프라 불요. |
| **5. Mainnet 2012~ 안정 운영** | KFIP 본선 후 즉시 메인넷 이전 가능. 미증명 신생 체인 의존 리스크 0. |

> **참고:** XRPL Hooks(스마트컨트랙트 like 기능)는 현재 Xahau 사이드체인에 배포, mainnet 미활성. 본 PoC는 Hooks 없이 EscrowCreate + EscrowFinish + crypto-condition만으로 구현 — 메인넷 즉시 작동 보장. Hooks 활성화 시 자동화 추가 가능 구조.
